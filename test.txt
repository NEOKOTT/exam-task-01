1) Чтобы перейти в директорию нужно написать команду cd, cd user01/docs
2) ~, -, - это абсолютый путь, тоесть путь по которому расположен файл или другая директория  
3) rm - remove, команда для удаления файла rm -r teachers
4) cd - команда для перемещения по директориям  cd user01/docs
5) cp - copy, команда для копирования файла или директории cp LDocs/* Lusers/Lstudent1
6) mv - move, mv Ldocs/LPhotos/L2023/file1.txt La.txt
7) mv file1.txt file2.txt
8) чтобы переместить нужно использоват комманду mv, mv Ldocs/LPhotos/L2023/file1.txt La.txt
9) флаг -R, рекурсирвно выводит дерево каталогов
10) флаг -r, выводит список файлов в обратном порядке
11) git - самая популярная в мире распределённая система контроля версий.
12) git init - инциализация нового репозитория
13) чтобы создать новую ветку нужно использовать git checkout -b users, создаться новая ветка и мы сразу перейдем в нее
14) чтобы сменить ветку нужно написать git checkout users, и мы перейдем в названную ветку
15) слияние это когда программисты объединяют две разных ветки в одну, в  которых они работали по отдельности, например каждый из них работал над
собственной фичей для проекта
16) git rm - позволяет удалять файлы из репозитория git
17) git revert -  создаёт новый коммит, который вносит изменения, противоположные указанному коммиту, по существу отменяя его.
